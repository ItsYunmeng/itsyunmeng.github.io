Linux_5.文件编辑器vim

Posted on 2020-06-02 In Linux

摘要: Vim常用操作👉Vim工作模式：命令模式(vi filename)，插入模式(a/i/o)，编辑模式(定位：:set no，: n；删除：x/nx,dd/ndd；复制和剪切：yy/dd p/P；替换和取消：r/R u；搜索和搜索替换：/string，n，%s or n1,n2s/old_string/new_string/g)，保存和退出(:wq，ZZ，:q!)。Vim使用技巧👉复制文件内容(:r 文件名)，导入命令执行结果(:r !命令)，定义快捷键(:map 快捷键 触发命令)，连续行注释(:n1,n2s/^/#/g)，替换(:ab mymail nordicorge@hotmail.com)。设置配置文件(vi /home/qiime2/.vimrc)👉在.vimrc中设置好。

Vim常用操作

Vim工作模式

命令模式: 进入，vi filename

插入模式: i/a/o开始输入内容，esc键退出。

  • a光标所在字符后插入
  • A光标所在行行尾插入
  • i光标所在字符前插入
  • I光标所在行行首插入
  • o光标下插入新行
  • O光标上插入新行

编辑模式: :开始输入命令，以回车结束运行

  • 定位命令
    • : set nu设置行号
    • : set nonu取消行号
    • : n到第n行
    • : $移至行尾
    • : 0移至行首
  • 删除命令
    • 按x键删除光标所在处字符
    • 按n(数字)+x键删除光标所在处后n个字符
    • 按n(数字)+dd键删除光标所在行，ndd删除n行
    • 按dG删除光标所在行到文件末尾内容
    • 按D删除光标所在行到行尾内容
    • 按: n1,n2d删除指定范围的行,(n1,n2)表示n2及之前的行都删除。
  • 复制和剪切命令
    • 按yy复制当前行
    • 按nyy复制当前以下n行
    • 按dd剪切当前行
    • 按ndd剪切当前以下n行
    • 按p和P粘贴在当前光标所在行下或行上
  • 替换和取消命令
    • 按r取消光标所在处字符
    • 按R从光标所在处开始替换字符，按esc键结束
    • 按u取消上一步操作
  • 搜索和搜索替换命令
    • 按/string搜索指定字符串
    • 按: set ic搜索时忽略大小写；按: set noic搜索时不忽略大小写
    • 按n键搜索指定字符串的下一个出现位置
    • 按: %s/old_string/new_string/g全文替换指定字符串
    • 按: n1,n2s/old_string/new_string/g在一定范围内替换指定字符串

保存和退出

  • 按:w保存修改
  • 按:w new_filename另存为指定文件
  • 按:wq保存修改并退出
  • 按ZZ快捷键，保存修改并退出
  • 按:q!不保存修改退出
  • 按:wq!保存修改并退出（文件所有者及root可使用）

Vim使用技巧

复制文件内容

把“文件名”中的内容复制到当前文件中

  • 用法：:r 文件名
  • 示例：:r /home/qiime2/lisi

导入命令执行结果

直接把命令执行的结果导入到当前的文件中

  • 用法：:r !命令
  • 示例：:r !date

定义快捷键

  • 用法：:map 快捷键 触发命令
  • 示例：:map ^P I#<ESC>，表示快捷键Ctrl+V+P代表跳到光标所在行的行首加一个#号之后退出。:map ^B 0x，表示快捷键Ctrl+V+B代表跳到光标所在行的行首然后删掉第一个字符。

连续行注释

示例

  • :n1,n2s/^/#/g在n1到n4行的行首加上#
  • :n1,n2s/#//g将n1到n4行的#替换为空
  • :n1,n2s/^#//g将n1到n4行行首的#替换为空

替换

示例

  • :ab mymail nordicorge@hotmail.com表示输入mymail后回车显示的是nordicorge@hotmail.com

设置配置文件

配置文件一般放在根目录或访客的目录下，文件名为.vimrc,只有在配置文件中设置好了，再创建文件的时候，输入快捷键或替换等才会统一。

  • 用法：vi /root/.vimrc或vi /home/qiime2/.vimrc
  • 用法：.vimrc中定义内容时不加冒号:，只写后面的内容。
