R包介绍及应用2|ggcor()，主quickcor() + Science相关性组合图

Posted on 2020-04-06 Edited on 2020-04-08 In R

摘要： 以ggcor 0.9.5进行演示。

简介

ggcor()

The goal of ggcor is to provide a set of functions that be used to visualize simply and directly a correlation matrix based on ‘ggplot2’.

quickcor()

快速地进行相关系数矩阵可视化的函数
quickcor(x, y, type=”full”, cor.test=T, cluster=T, show.diag=F, legend.position=”right”, method=”pearson”)

  • 数据(x,y)：可以只有x，若x和y同时存在，x在纵轴显示，y在横轴显示。
  • 方形图或三角形图：type=’full’(默认),upper’,’lower’,…
  • 显著性检验：cor.test=T，
  • 相关性求取的算法：method=’kendall’(默认),’pearson’,’spearman’,…
  • 只支持对称相关系数矩阵聚类：cluster=T
  • 对角线：show.diag = FALSE(默认), TRUE
  • 图例的位置：legend.position=”right”
  • 图层
    • 颜色(颜色表示相关性)；形状(颜色和大小均表示相关性)： geom_colour(); geom_circle2(),geom_ellipse2(),geom_pie2(),geom_ring(),geom_square(),geom_star(),…
    • 相关性(r)数值：geom_number(aes(num=r))
    • 显著性：geom_mark(r=NA,sig.thres=0.05,size=1.5,colour=”grey90”),r=NA不显示数值，仅显示星号，若不设置此，既显示数值也显示星号。
    • 用来标记非显著性：geom_cross()、
    • 改变填充颜色（需要先加载ggplot2)：scale_fill_gradientn(colours = c(“red”, “white”, “blue”))，red是1，blue是-1，可调换。
    • 对角线添加标签：add_diag_label(size = 5, colour = “red”)
    • 移除坐标轴标签：remove_axis()
    • geom_abline(slope = -1, intercept = 12)

    • geom_cross(),排除不显著相关

    • geom_confbox(),展示置信区间

    • mantel test：mantel_test(x, y,
